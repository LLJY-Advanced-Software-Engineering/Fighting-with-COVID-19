About the data

Gradually Change

This data updates from various APIs , might not reflect some cases still being reported. It is possible that some changes from sources produce imprecise / no data being reported for the several regions.

Official Statistics

The numbers are based on official estimates and test. There are some deviation compared with the real situations. For example, some countries that lack the proper infrastructure to diagnose patients, unconfirmed cases may not be resolved.
